# Diagrama de Casos de Uso

## Ator Principal

O principal ator do sistema é o **Usuário**, que representa qualquer pessoa que utilize o sistema para gerenciar suas tarefas.

## Casos de Uso

| Caso de Uso          | Descrição |
|---------------------|-----------|
| Cadastrar tarefa    | Permite ao usuário criar uma nova tarefa no sistema |
| Listar tarefas      | Permite visualizar todas as tarefas cadastradas |
| Atualizar tarefa    | Permite editar informações de uma tarefa existente |
| Remover tarefa      | Permite excluir uma tarefa do sistema |

## Descrição dos Casos

### Cadastrar tarefa
O usuário informa os dados da tarefa e o sistema armazena as informações.

### Listar tarefas
O sistema apresenta ao usuário a lista de tarefas cadastradas.

### Atualizar tarefa
O usuário pode alterar os dados de uma tarefa existente.

### Remover tarefa
O usuário remove uma tarefa que não seja mais necessária.
