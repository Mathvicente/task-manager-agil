# Diagrama de Classes

## Classe: Tarefa

| Atributo | Tipo | Descrição |
|--------|-----|----------|
| id | int | Identificador único da tarefa |
| titulo | string | Título da tarefa |
| descricao | string | Descrição detalhada |
| status | string | Estado da tarefa (To Do, In Progress, Done) |

## Métodos

| Método | Descrição |
|-------|----------|
| criar_tarefa() | Cria uma nova tarefa |
| listar_tarefas() | Retorna todas as tarefas |
| atualizar_tarefa() | Atualiza dados de uma tarefa |
| remover_tarefa() | Remove uma tarefa do sistema |
